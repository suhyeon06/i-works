## Iworks :chart_with_upwards_trend:

각 부서 별로 볼 수 있는 게시판, 할 일을 생성하고 캘린더로 일정을 관리, OpenVidu를 활용한 화상 미팅룸, 부서 사람들의 주소록 등을 통해 업무의 효율성을 극대화 할 수 있는 업무 협업 도구입니다.
